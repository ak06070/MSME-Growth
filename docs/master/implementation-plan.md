# Master Implementation Plan

## Delivery stages
1. Repository bootstrap
2. Foundation services
3. Auth, RBAC, and tenancy
4. Finance data model
5. CSV invoice ingestion
6. Collections workflows
7. Cashflow summaries
8. Loan-readiness workspace
9. AI recommendation layer
10. Observability, security hardening, release prep

## Delivery cadence
- Build in bounded specs.
- One spec = one branch.
- Review after each spec.
- No parallel feature expansion before foundation is stable.

## Release strategy
- Internal alpha
- Pilot with limited MSMEs
- Controlled production rollout
