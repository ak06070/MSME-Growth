# Spec 000 - Repository Bootstrap

## Purpose
Create the baseline monorepo structure and engineering standards required for all future development.

## In scope
- Monorepo folder structure
- Shared README placeholders
- Linting and formatting setup
- Type-checking setup
- Test runner setup
- Basic CI pipeline
- Environment variable strategy
- Base app shells for web and api

## Out of scope
- Business features
- Auth
- Database models beyond placeholders
- Integrations
- AI logic

## Acceptance criteria
- Repository structure matches approved layout
- CI runs lint, type-check, and tests
- Web and API apps boot with placeholder routes/pages
- Local setup instructions exist
- No business domain logic added
